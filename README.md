## JJK — текстовый бой

Это статическая HTML-страница (без сборки и зависимостей).

### Как запустить

- Открой `index.html` двойным кликом  
  или
- Запусти любой простой локальный сервер (чтобы всё работало одинаково в браузерах).

### Структура

- `index.html` — разметка + основной CSS/JS
- `styles.css` — небольшой слой дополнительных улучшений UI (override поверх inline-стилей)
